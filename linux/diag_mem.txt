lib和进程内存使用
procrank  查看进程所占用的内存 
	VSS：Virtual Set Size，虚拟耗用内存
	RSS：Resident Set Size，实际使用物理内存。它包含了所有该进程使用的共享库所占用的内存
	PSS：Proportional Set Size，实际使用的物理内存，它与RSS不同，它会按比例分配共享库所占用的内存。
	USS：Unique Set Size，进程独自占用的物理内存。这部分内存完全是该进程独享的
